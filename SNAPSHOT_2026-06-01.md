### 廊亭 (loggia) debug 记录
- **Bug 1**: async generator + sync Claude blocking → SSE 不 flush 给浏览器
  - 原因: `async def` generator 里 sync `_client.messages.stream()` 阻塞事件循环,
    但 `await asyncio.sleep(0.5)` 在中间导致 Starlette 的 StreamingResponse 无法正确迭代
  - 修复: 改成 `def` (sync generator), StreamingResponse 自动在 threadpool 跑,flush 正常
  - curl 始终正常(curl 不依赖 event loop flush),浏览器才暴露

- **Bug 2**: 前端 `insertBefore(label, el)` 在 el 还没 append 到 wrap 之前调用
  - 原因: DOM API 要求 referenceNode 已经是 parentNode 的子节点
  - 表现: 整个气泡静默不渲染,没有 JS 报错(因为 try/catch 吞了)
  - 修复: 先 appendChild(label), 再 appendChild(el)

- **Bug 3**: V 气泡用了珩的颜色
  - 原因: `.char-v` 只设了左边框没设背景色,继承了默认 assistant 样式
  - 修复: `.char-v` 加 `background: var(--v-bubble)`

- **Bug 4**: service worker 缓存旧 HTML,新功能不生效
  - 修复: SW 改 network-first for HTML, cache-first for static assets; 版本号递增强制更新
